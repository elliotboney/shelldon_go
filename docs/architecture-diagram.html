shelldon Architecture

A digital pet — LLM-free supervised core, isolated worker brain, hybrid memory, dream-cycle learning

Core — LLM-free (depguard enforced)

  • • Bus routes typed Envelopes by Kind
  • • Dispatch runs the turn loop + records convo
  • • Arbiter gates ≤1 worker turn in flight (RAM bound)
  • • Reflexes (blink, mood drift) need no LLM
  • • Sole writer to every store

Trust & Brain

  • • Broker is the only credential holder
  • • Provider chain: GLM → Ollama → OpenAI → Anthropic
  • • No secrets ever travel on the bus
  • • Worker untrusted, behind a swappable interface
  • • M3: worker moves to a uid-separated subprocess

Memory & Learning

  • • history.db: SQLite + FTS5, messages + learnings
  • • capture_learning on the turn hot-path
  • • Curated markdown: DIRECTIVE.md, about.md, facts/
  • • Dream cycle (~6h) promotes/prunes learnings
  • • Memory-augmented prompts ground each reply
